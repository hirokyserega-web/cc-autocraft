# cc-autocraft

Распределённая система автоматического крафта для **ComputerCraft: Tweaked**.
Центральный компьютер (Core) управляет складом, базой рецептов и очередью задач;
черепахи-воркеры исполняют крафт через сундуки-буферы.

## Возможности
- **Новый UI**: Современная боковая панель с визуальным откликом на нажатия.
- **Авто-крафт по дереву зависимостей**: система сама крафтит промежуточные предметы.
- **Запись рецептов через тест-крафт**: выкладываете крафт в сундук-сканер, система крафтит один раз и сохраняет рецепт.
- **Масштабируемость**: Поддержка множества черепах-воркеров одновременно.

## Установка
Запустите одну команду в Minecraft на Core-компьютере (требуется HTTP API):
```bash
wget run https://raw.githubusercontent.com/hirokyserega-web/cc-autocraft/main/install.lua
```
Установщик скачает все файлы и перезагрузит компьютер. После перезагрузки система запустится автоматически.

## Настройка
1. **Core**: Подключите к проводной сети сундуки-склады, сундук-сканер и монитор. Установите беспроводной модем.
2. **Воркеры (черепахи)**: У каждой черепахи — беспроводной модем и два соседних сундука (входной и выходной буферы). Запустите `main.lua run` (черепаха сама поймет, что она воркер).
3. На Core запустите `main.lua run`.

## Как пользоваться
1. **Настройки**: Выберите сундук, который будет служить сканером рецептов.
2. **Рецепты**: Положите ингредиенты в сканер (сетка 3x3), нажмите "ТЕСТ КРАФТА", затем "СОХРАНИТЬ".
3. **Склад**: Находите нужные предметы и заказывайте их крафт в один клик.
