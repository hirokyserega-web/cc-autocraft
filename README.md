# cc-autocraft

Распределенная система автоматического крафта для ComputerCraft: Tweaked.

## Особенности
- **Масштабируемость**: Поддержка от 1 до 10+ черепах-воркеров.
- **Отказоустойчивость**: Сохранение очереди и состояния при перезагрузке.
- **Умное резервирование**: Предотвращение конфликтов ресурсов.
- **Buffer Mode**: Работа через сундуки-буферы для максимальной совместимости.

## Установка
Запустите следующую команду в Minecraft (требуется HTTP API):
```bash
wget https://raw.githubusercontent.com/hirokyserega-web/cc-autocraft/main/install.lua
install.lua
```

## Настройка
1. Подключите Core к проводной сети с инвентарями и монитором.
2. Установите черепахи с беспроводными модемами.
3. Запустите `core/server.lua run` на сервере.
4. Запустите `worker/worker.lua run` на черепахах.
