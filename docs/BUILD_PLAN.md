# План разработки cc-autocraft

## Этап 1: Инфраструктура — ГОТОВО
- [x] Инициализация репозитория.
- [x] ARCHITECTURE.md и BUILD_PLAN.md.
- [x] Библиотеки: util (включая надёжное определение инвентарей),
      net, itemmatch, lang.

## Этап 2: Фундамент (Core) — ГОТОВО
- [x] `lib/util.lua`: сериализация, логи, `isInventory`/`getInventories`
      (определение по наличию методов `list`/`size` — работает на всех
      версиях CC:T, в отличие от `getType=="inventory"`).
- [x] `lib/itemmatch.lua`: сравнение по ID/тегам/NBT.
- [x] `lib/net.lua`: обёртка над Rednet.
- [x] `core/storage.lua`: инвентаризация и резервирование; сундук-сканер
      и буферы исключаются из склада.

## Этап 3: Рецепты и Планировщик — ГОТОВО
- [x] `core/recipes.lua`: загрузка/сохранение, чтение центральной сетки 3×3,
      авто-раскладка (левый блок → центр), `has`/`list`/`add`/`remove`.
- [x] `core/planner.lua`: дерево крафта с группировкой одинаковых
      ингредиентов, расчётом партий по `output_count`, резервированием
      сырья, и `check()` — проверка возможности без резервирования.

## Этап 4: Диспетчер и Воркер — ГОТОВО
- [x] `core/dispatcher.lua`: раздача задач, авто-назначение буферов,
      сериализация плана (`isReady`: родитель ждёт дочерние задачи).
- [x] `worker/worker.lua`: FSM черепахи, маппинг центральной сетки 3×3
      входного сундука на крафтовую сетку черепахи.

## Этап 5: Финализация Core — ГОТОВО
- [x] `core/server.lua`: главный цикл, выбор сканера (с сохранением),
      тест-крафт и сохранение рецепта, обработка кнопок UI.

## Этап 6: UI и Инсталляция — ГОТОВО
- [x] `ui/monitor.lua`: переработанный интерфейс (ГЛАВНАЯ/СКЛАД/РЕЦЕПТЫ/
      НАСТРОЙКИ), живой предпросмотр сетки, модалки заказа/теста/ошибки,
      индикация «есть/нужно» и возможности изготовления.
- [x] `install.lua`: скрипт установки с очисткой и авто-запуском.

## Этап 7: Тестирование — ГОТОВО
- [x] Юнит-тесты логики (planner, storage, recipes, dispatcher) под Lua 5.3
      с моком окружения CC:T — 10/10 проходят (`test/run_tests.lua`).
- [ ] Приёмка в реальном Minecraft (требует игры).
