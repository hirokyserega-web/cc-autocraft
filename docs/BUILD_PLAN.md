# План разработки cc-autocraft

## Этап 1: Инфраструктура и Проектирование (ТЕКУЩИЙ)
- [x] Инициализация репозитория.
- [x] Написание ARCHITECTURE.md и BUILD_PLAN.md.
- [ ] Базовые библиотеки (util, net, itemmatch, lang).

## Этап 2: Фундамент (Core)
- [ ] `lib/util.lua`: Сериализация, логирование, работа с файлами.
- [ ] `lib/itemmatch.lua`: Сравнение предметов по ID, тегам и NBT.
- [ ] `lib/net.lua`: Обертка над Rednet.
- [ ] `core/storage.lua`: Инвентаризация и резервирование.

## Этап 3: Рецепты и Планировщик
- [ ] `core/recipes.lua`: Хранение и менеджмент рецептов.
- [ ] `core/planner.lua`: Расчет дерева крафта и необходимых ресурсов.

## Этап 4: Диспетчер и Воркер
- [ ] `core/dispatcher.lua`: Раздача задач, управление буферами.
- [ ] `worker/worker.lua`: FSM черепахи (idle, loading, crafting, unloading).

## Этап 5: Финализация Core
- [ ] `core/server.lua`: Главный цикл.
- [ ] `core/machines.lua`: Работа с механизмами.

## Этап 6: UI и Инсталляция
- [ ] `ui/*`: Интерфейс на мониторах.
- [ ] `install.lua`: Скрипт установки.

## Этап 7: Тестирование и Приемка
- [ ] Проверка по 11 критериям в CraftOS-PC.
